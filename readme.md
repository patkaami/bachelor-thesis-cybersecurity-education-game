### Educational Application for Working with Digital Technologies ###

### Launch Instructions: ###

Open the file "Aplikácia.exe" or run "main.py" (requires installation of pygame, zxcvbn, re libraries)

### Game Description and Content: ###
The game serves as an educational tool on the topic of information security/working with digital technologies.
It contains a menu with 3 categories, an information icon containing the rules, and an arrow icon that serves as a restart for already completed categories:

### 1. Sensitive Data
Accompanying text
- **Task 1:**  
  Different scenarios appear on the screen, and the player decides whether sharing this data can endanger/limit them or not. If so, they select possible risks for the given scenario. Subsequently, an explanation is displayed as to why the given data can/cannot be shared.
- **Task 2:**  
  The player sorts data into those that are safe to share with strangers on the internet and those that are not.
---

### 2. Passwords
- **Task 1:**  
  The player marks all correct answers regarding the properties of a strong password.
- **Task 2:**  
  Sorting passwords from weakest to strongest.
- **Task 3:**  
  The player meets a "hacker" in a password creation simulator. The hacker tries to guess their password while the screen displays how long it would take to crack the password and possible improvements. If the player creates a sufficiently strong password, they successfully solve the task.

---

### 3. Phishing
Accompanying text
- **Task 1:**  
  The player catches fish. After clicking on a fish, a phishing message is displayed. The player must identify all suspicious elements. If they correctly reveal all of them, the caught fish is displayed at the top of the screen. After catching all fish, they proceed to the next task.
- **Task 2:**  
  Sorting statements about phishing into correct or incorrect columns.
